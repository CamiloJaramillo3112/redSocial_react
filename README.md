# redSocial_react
Proyecto académico en React utilizando Vite, W3.CSS y FontAwesome. Simula una red social con componentes dinámicos (Navbar, Sidebar, Feed, Eventos, Solicitudes de amistad y Footer) organizados de forma modular.

# Proyecto Red Social en React

Aplicación web estilo red social desarrollada con **React** y **Vite**, utilizando **W3.CSS** y **FontAwesome** para el diseño.  
Convierte una plantilla estática en una interfaz dinámica y modular, organizada en componentes reutilizables.

## 🚀 Características
- Barra de navegación (Navbar)
- Sidebar izquierda con perfil e intereses
- Feed dinámico con publicaciones
- Sidebar derecha con eventos y solicitudes de amistad
- Footer con créditos

## 🛠️ Tecnologías
- React + Vite
- W3.CSS
- FontAwesome
